# Simply Appe

This repository contains a custom theme for ITGmania, created as a mashup of three different themes:

*   **Simply Love** (Original)
*   **Simply Love zmod** (developed by Zarzob and Zankoku)
*   **In The Groove 2.1** (developed by Dando)

## Main Changes

The primary goal of this mashup is to combine the best features of the source themes:

*   **Visual UI**: Adopts the visual user interface style from **In The Groove 2.1**.
*   **Mods & Features**: Incorporates the gameplay modifications and quality-of-life features from **Simply Love zmod**.
*   **Custom Tweaks**: Includes various UI adjustments and refinements tailored to personal preferences.
